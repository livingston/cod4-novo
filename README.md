## CoD4 Novo

Call of Duty 4 library of scripts for servers.

### Requirements:
`CoD4X 1.7a`

### Installation:
* Place both `novo` and `maps` folders into your `/main_shared/` folder
* Start the server

## Features:

* Geolocation based Welcome
